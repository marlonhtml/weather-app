# Weather App

A simple desktop weather application built with Python and Tkinter. Get real-time weather information for any city with a clean GUI interface.

## Quick Start

1. Install dependencies:
```bash
pip install requests python-dotenv
```

2. Get a free API key from [WeatherAPI.com](https://www.weatherapi.com/)

3. Create a `.env` file:
```
api_key=your_api_key_here
```

4. Run the app:
```bash
python app.py
```

## Features

- Search any city worldwide
- Current temperature, wind speed, and conditions
- Dynamic day/night theme
- Clean GUI interface

## Tech Stack

Python • Tkinter • WeatherAPI.com
